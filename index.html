  • 001

    NoEmmm

    一款基于日历的待办事项管理应用，可以针对具体的日子添加和删除待办事项，同时支持注册登陆和云同步。由于基于React，所以大量采用了ES6语法，用了moment.js对日期进行处理，使得日历部分的代码更便于管理。遗憾的是目前只支持桌面端，后续还会加入对移动端的支持。

  • 002

    在线简历生成器

    写简历好麻烦，所以有了这么一个简历生成器，可以动态生成一份样式不错的简历。支持注册和登陆，登录之后还可以支持在线预览。项目主要是基于VUE，用VUEX做了状态管理，样式方面是用了scss，使得样式在后期维护上可以更加直观。期间，有人反映在不同分辨率的电脑上布局完全走样。最后决定将所有单位改成vw来解决这个问题。

    技术栈： VUE + VUEX + SCSS+ webpack

  • 003

    NetEaseMusic

    整个项目用jQuery实现，由于jQuery在DOM操作上相较于原生js确实高效很多。css样式初次尝试用scss，不得不感叹一句SCSS大法好，样式一目了然。不过也碰到了一些问题，比如歌曲暂停后重新播放旋转的光盘会被重置，后来通过animation-play-state控制动画的播放和暂停解决了这个问题。还有由于搜索框监听input事件，只要一有输入就有ajax请求，会导致请求过多，影响最后的结果，最终通过debounce函数解决了这个问题。

    技术栈： jQuery + webpack + SCSS

  • 005

    豆瓣租房爬虫

    针对杭州租房豆瓣小组的爬虫，nodejs部分用axios发送请求，再用cheerio对返回的结果进行处理。主要得益于es7的async/await，使得代码整体逻辑顺畅。最后顺手做了前端页面的展示，用express搭了服务端，不得不佩服，express确实还挺好用的。唯一的遗憾就是豆瓣对图片外链的限制，前端展示上没办法展示图片。

    部署： Node + Express

    技术栈： jQuery + Nodejs + Cheerio + axios + Express
